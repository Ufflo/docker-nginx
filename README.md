Small and simple Nginx with volumes

## Volumes

* **/etc/nginx**

  Configuration files

* **/var/log/nginx**

  Log files

* **/var/www/html**

  Data files

## Ports

* **80 TCP**

  HTTP

* **443 TCP**

  HTTPS
